# تحديث أيقونة التطبيق (Windows)

لضمان ظهور الأيقونة بشكل صحيح على سطح المكتب وشريط المهام في ويندوز، اتبع الخطوات التالية:

1. **تحويل الأيقونة إلى صيغة .ico**
   - استخدم أي أداة تحويل صور (مثل https://cloudconvert.com/png-to-ico) لتحويل ملف `icon.png` إلى `icon.ico`.
   - ضع ملف `icon.ico` في المجلد الرئيسي للمشروع.

2. **تحديث إعدادات package.json**
   - تأكد أن خاصية `icon` في قسم `win` و`nsis` تشير إلى `icon.ico` وليس `icon.png`.

3. **إعادة بناء التطبيق**
   - شغّل الأمر:
     ```
     npm run build-win
     ```
   - بعد التثبيت، ستظهر الأيقونة الجديدة على سطح المكتب وشريط المهام.

---

## ملاحظات
- يفضل دائماً استخدام صيغة `.ico` لأيقونات ويندوز.
- إذا واجهت أي مشكلة في ظهور الأيقونة، تأكد من مسار الملف وصيغته.
